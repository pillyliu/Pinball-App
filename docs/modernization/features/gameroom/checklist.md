# GameRoom Checklist

Use this list before marking any GameRoom modernization step complete.

- Home layout matches the written spec.
- Collection toggle labels and behavior match the written spec.
- First tap selects and second tap opens.
- Settings sections appear in the written order.
- Machine View segments and behavior match the written spec.
- Input actions map to the same data model behavior on both platforms.
- Log edit/delete behavior matches on both platforms.
- Archive filters and open behavior match on both platforms.
- Pinside import match/review/import behavior matches on both platforms.
- Library overlay behavior matches on both platforms.
- Known drift, if any, is recorded in `ledger.md`.
